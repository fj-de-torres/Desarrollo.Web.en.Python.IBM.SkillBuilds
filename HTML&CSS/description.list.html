Definiciones importantes

FTP

Protocolo para la transmisión de ficheros entre ordenadores.

HTML

Lenguaje de etiquetas empleado para generar páginas web

PHP

Lenguaje interptado en servidor que permite general páginas web dinámicas

No considerado un lenguaje de programación porque no tiene la capacidad de generar ni bifurcaciones ni bucles.
